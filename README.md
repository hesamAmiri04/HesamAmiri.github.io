# Hesam Amiri — Personal Portfolio

> Cloud · Edge · IoT Research

A minimal, dark-themed personal portfolio for Hesam Amiri — MSc Telecommunications Engineering candidate at the University of Bologna.

## 🚀 Deploy to GitHub Pages

1. Upload the contents of this folder to a GitHub repository named `<your-username>.github.io`
   - e.g. `hesamAmiri04.github.io`
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**

Your site will be live at: `https://hesamAmiri04.github.io`
