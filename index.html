Research

WiLab — Wireless Communication Laboratory · UniBO · Supervisor: Prof. Chiara Buratti

Ongoing · 2024 — present

RL-based dynamic MAC for IoT networks

Static CSMA wastes airtime when the channel is busy and starves nodes when it isn't. I'm replacing it with a reinforcement-learning agent that decides when to transmit based on what it actually observes on the channel — no fixed scheduling, no a-priori traffic model. The agent runs on real IEEE 802.15.4 nodes in the WiLab testbed, so it sees the real losses, the real interference, the real hardware quirks. Python implementation, on-device evaluation.

  • HARDWAREIEEE 802.15.4 testbed
